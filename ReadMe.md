# Visual Studio Extensibility Samples
「[Fukuoka.NET Conf 2017 〜ハンズオンで体験する最新.NETテクノロジー 〜](https://fukuten.connpass.com/event/63612/)」で解説した Visual Studio Extensibility のサンプルです。

- HelloVsCommand
  - カスタムコマンドのサンプルです
- HelloVsToolWindow
  - カスタム Tool Window のサンプルです
    - UserControl プロジェクトで利用するアイコンは、[MahApps.Metro.IconPacks](https://github.com/MahApps/MahApps.Metro.IconPacks) に厳密名を付与したものを利用しています。
- HelloVsTemplate
  - カスタムプロジェクトテンプレートのサンプルです
